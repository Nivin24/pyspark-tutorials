# Spark SQL

## Overview
This folder contains tutorials on using Spark SQL for querying and analyzing data using SQL syntax.

## Topics Covered
1. **01_spark_sql_basics.py** - SQL fundamentals with Spark
2. **02_creating_views.py** - Temporary and permanent views
3. **03_sql_queries.py** - Complex SQL queries
4. **04_optimization.py** - Query optimization techniques

## Key Concepts
- Temporary views
- Permanent views
- SQL syntax in Spark
- Catalyst optimizer
- Query execution plans

- Basic SQL understanding
